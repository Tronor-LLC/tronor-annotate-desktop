## What is T-Annotate?

T-Annotate is a lightweight screen annotation tool that lets you draw on top of any application. It runs as a menu bar app and overlays transparent canvases on all your displays, allowing you to annotate anything on screen using ink brushes.

Key features:

- **Multi-display support** — draw across all connected displays
- **Ink brushes** — Pressure Pen, Pen, Calligraphy, and Highlighter with adjustable size
- **6 customizable colors** — quick-select palette with full hex color picker
- **Background overlay** — adjustable opacity and color for better contrast
- **Floating toolbar** — draggable annotation toolbar for quick access to tools
- **Global hotkeys** — start/stop annotation mode from any application
- **Menu bar app** — stays out of your way, accessible from the status bar
- **Launch at Login** — optional auto-start on reboot
- **Dark & Light mode** — adapts to your macOS appearance
